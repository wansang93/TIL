## EUV 공정

옷을 만들 때: 디자이너의 도면 -> 제작

반도체 만들 때: 설계 회사 -> 설계도 -> 칩 제작

- **Fabless**: 설계하는 사업
- **Foundary**: 생산하는 사업
- **Disign House**: 설계 최적화 역할

반도채를 생산 때 반도체 원판 위에 설계도대로 빛을 통해 사진을 찍듯이 찍어냄

이 때 빛의 종류가 **EUV**(Extreme Ultra Violet: 극 자외선) 입니다.

파장이 짧아 정교한 작업 가능

## 메모리 반도체와 시스템 반도체

- 메모리 반도체: 정보 저장하는 반도체
- 시스템 반도체: 연산 명령하는 반도체
- 비메모리 반도체: 한국에서만 쓰는 용어, 한국의 메모리 시장이 워낙 커서 메모리가 아닌 부분의 반도체, 즉 시스템 반도체

## 시스템 반도체의 중요성

기계의 지능화에서 판단하는 반도체가 미래의 핵심 기술, 삼성전자가 지원을 합니다.

## 이미지 센서 산업

- 이미지 센서: 필름, 망막의 역할
- 포토다이오드(photodiode): 빛을 전기로 바꿔주는 센서
- 픽셀: 한 점
- 해상도: 픽셀로 이루어진 이미지

스마트 폰에 이미지 센서가 들어가야 함

- 이미지 센서가 크면서 픽셀이 작으면 좋은 사진
- 픽셀이 작음 -> 픽셀 수가 많음 -> 화소 좋아짐

## 픽셀이 작아지면 문제점

- 간섭현상: 이미지 센서는 색을 정확하게 재연이 중요, 빛이 들어오면 다른 센서도 건드려서 빛이 번짐
  - 해결: 아이소셀(Isolate + cell) 플러스 기술 -> 칸막이 기술, 다른 센서에 못건들이게 칸막이를 침, 칸막이가 개좋음
- 어두울 때 촬영이 힘듬: 사진이 자글자글 한 경험의 이유, 빛을 획득하지 못해서
  - 해결: 테트라셀(Tetracell) -> 4개의 픽셀을 하나로 묶어 빛을 더 잘 인식하게 하는 기술
