# [220118(Tue)] Java Basic2

## Java Basic(OOP, )

### ✨ 01. 객체지향 프로그래밍이란

**Object Oriented Programming**, 현실세계가 객체지향이다.

- 신뢰성이 높음
- 추가/수정/삭제 용이
- 재 사용성이 높음

### ✨ 02. JVM의 메모리 구조

class area, method stack, heap 로 구성

1. class area
   - Field 정보, Method 정보, Type 정보, 상수 풀
2. method stack
   - **thread 별로 별도 관리**
   - 메서드 호출 순서대로 쌓이는 구조
   - 메서드 프레임에 로컬변수도 쌓이는 구조
   - 로컬변수는 선언된 영역을 벗어나면 삭제
3. heap
   - **thread에 의해 공유**
   - 객체를 저장하기 위한 영역
   - 생성된 객체 유저가 삭제 불가, GC만 제어 가능
