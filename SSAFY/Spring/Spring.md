# Spring

## Q1. 스프링왜씀?

- 자바 써서 웹 앱 개발할때 거의 표준급 프레임워크여서
- JSP, MyBatis, JPA등 다양한 기술도 있긴 하지만

## Q2. 스프링 주요 특징 머임?

- 의존 주입(DI: Dependency Injection) 지원
- AOP(Aspect-Oriented Programming) 지원
- MVC 웹 프레임워크 제공
- JDBC, JPA 연동, 선언적 트랜잭션 처리, DB 연동 지원
- 스케줄링, JMS, 이메일 발송, 테스트 지원 등 다양한 기능 제공

## Q3. 스프링 주변 프로젝트는 머머 있음?

- 스프링 데이터, 스프링 시큐리티, 스프링 배치, 스프링 하둡, 스프링 소셜 등 많음
