### String, StringBuffer, StringBuilder

String

- 단점: 문자 수정되면 새로운 객체를 생성해서 참조값을 바꿈
- 문자열 연산이 적고 멀티스레드 환경일 경우

StringBuffer

- 문자열 연산이 많고 멀티스레드 환경일 경우
- 같이쓰는 화장실 -> 협업할때 사용
- 장점: 동기화 지원(메서드별 synchronized keyword가 존재)

StringBuilder

- 문자열 연산이 많고 단일스레드이거나 동기화를 고려하지 않아도 되는 경우
- 혼자쓰는 화장실 -> 빠름(노크 ㄴㄴ해)

|              |    String     | StringBuffer | StringBuilder |
| :----------: | :-----------: | :----------: | :-----------: |
|   Storage    |  String pool  |     Heap     |     Heap      |
|  Modifiable  | No(immutable) | Yes(mutable) | Yes(mutable)  |
| Thread safe  |      Yes      |     Yes      |      No       |
| Synchronized |      Yes      |     Yes      |      No       |
|  Performace  |     Fast      |     Slow     |     Fast      |

### N*M 배열 4방 탐색

```java
package com.ssafy.array;

import java.io.FileInputStream;
import java.io.FileNotFoundException;
import java.util.Scanner;

/* Input
2
4 5
1 2 3 4 5
11 12 13 14 15
5 6 7 8 9
15 16 17 18 19
1 1
3 3
1 2 3
4 5 6 
7 8 9
2 2
*/

public class Deriction4Test2_Scanner {

public static void main(String args[]) throws FileNotFoundException {
    
    System.setIn(new FileInputStream("input.txt"));
    Scanner sc = new Scanner(System.in);
    int T = sc.nextInt();
    for (int tc = 1; tc <= T; tc++) {

        int N = sc.nextInt();
        int M = sc.nextInt();

        int[][] arr = new int[N][M];

        for(int i=0; i < N; i++) {
            for(int j=0; j < M; j++) {
                arr[i][j] = sc.nextInt();
            }
        }
        
        int x = sc.nextInt();
        int y = sc.nextInt();
        
        int nx, ny;
        int[] dx = {-1, 1, 0, 0};
        int[] dy = {0, 0, -1, 1};
        
        for (int d = 0; d < dx.length; d++) {
            nx = x + dx[d];
            ny = y + dy[d];
            if (nx >= 0 && nx < N && ny >= 0 && ny < N) {
                arr[nx][ny] = 0;
            }
        }
        
        System.out.println("#" + tc);
        print(arr);
    }
    
    sc.close();

}

private static void print(int[][] arr) {
    for (int i = 0; i < arr.length; i++) {
        for (int j = 0; j < arr[i].length; j++) {
            System.out.printf("%3d", arr[i][j]);
        }
        System.out.println();
    }
}

}
```
